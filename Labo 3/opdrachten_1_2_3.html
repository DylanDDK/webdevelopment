    Opdracht 3

    De head bestaat uit een verschillende meta's, een title, verschillende links scripts e styles. Als we dan kijken naar de body zien we dat deze bestaat uit een header, verschillende div's, een nav, een footer en een script.

    Als we op een gesloten sectie klikken zien we meer in detail wat er allemaal in het element staat.

    Al de extra toevoegingen zijn er bij gekomen zodat de code ook bruikbaar wordt zonder gebruik te maken van JavaScript.
